실시간 개별 페이지 로드 시간
실시간으로 실제 고객의 성능을
확인하고 빠르게 대응하세요.
제니퍼 프론트의 실시간 대시보드를 통해 지금 발생하는 상황을 관전하고 문제 발견 즉시
실시간 개별 페이지 로드 시간 차트를 통해 해결할 수 있습니다.

개별 페이지 로드 시간은 모든 트랜잭션의 페이지 로드 시
간을 개별 점 그래프로 표현한 차트입니다.
개별 페이지 로드 시간의 문제 사항을 드래그하면 원인을
빠르게 파악할 수 있는 타임라인 등의 기능을 제공합니다.
